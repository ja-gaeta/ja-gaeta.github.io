O Ponto G++ e eu

Comecei a desenvolver este site ainda experimentalmente, enquanto aprendia HTML, CSS, PHP, JavaScript e outras ferramentas para editoração na Web, servindo como um laboratório para testar as diversas técnicas a medida em que as ia dominando. Posteriormente, conheci diversos Sistemas de Gerenciamento de Conteúdo (CMS - do inglês Content Management System), como Drupal, Modx, Wordpress e, finalmente, geradores de páginas estáticas como Hugo, Jekyll, Grain e JBake, este último desenvolvido na linguagem Java, passando a dispor do conhecimento e ferramentas necessárias para publicar um blog pessoal, no qual pudesse divulgar assuntos relacionados com os hobbies que mantive ao longo da vida.

Os assuntos que você encontrará aqui são relacionados, primeiramente, com a programação de computadores, que venho estudando há muitos anos, desde que adquiri meu primeiro computador pessoal TK-85, ainda no final da década de 1980. Passei pelo Basic, dBase(s), C, C++, Delphi, C#, Java, JavaScript, HTML5, CSS3 tudo aprendido sozinho, como hobby mesmo. Acho que isso é um estímulo para aqueles milhares de jovens Geeks que estão entrando agora no mercado de trabalho. Com o acesso ao enorme cabedal de conhecimento disponibilizado pela Internet ao alcance das mãos, não há limites para até onde um indivíduo pode chegar com o esforço próprio.

Uma vez que precisei me aprofundar bastante em desenvolvimento para Web afim de poder publicar este blog, irei abordar também HTML, CSS e outros temas correlatos com a matéria.

Outro assunto que focalizarei é a mecatrônica, versão moderna do meu antigo hobby de eletrônica, praticada desde os tempos de radioamadorismo, no princípio da década de 1980. Possuo todas essas novidades: Raspberry Pi e Arduíno. Como dizem, a gente nunca larga os brinquedos, eles só vão ficando cada vez mais caros...

Aproveitarei para incluir também conteúdo sobre vários programas fantásticos de código aberto que venho usando para desenvolvimento gráfico, como Gimp e Blender e, finalmente, não faltará espaço para outra velha paixão: o sistema operacional Linux. Atualmente aposentado, sobra-me tempo para dedicação integral a esses hobbies, compartilhando tudo o que achar interessante aqui no Ponto G++, o ponto de encontro de velhos (e novos) Geeks.

Grato pela visita. Espero que aproveite o material e volte sempre.
