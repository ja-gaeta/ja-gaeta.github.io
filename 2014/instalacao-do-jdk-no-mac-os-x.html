Instalação do JDK no Mac OS X

  1. programação
  2. java
  3. tutoriais

 30 December 2014    J.A. Gaeta Mendes    

Forneço neste artigo um passo-a-passo para instalação do JDK/JRE (Java SE 8) no Mac OS X. Para outras versões do JDK/JRE, consulte a documentação do seu sistema operacional e/ou a documentação da Oracle, caso encontre alguma dificuldade.

1º Passo - Download

Efetue o download do arquivo executável de instalação no site da Oracle, neste link. Selecione JDK Download. Ao instalar o JDK, o JRE correspondente também é instalado. Observe a imagem a seguir:

Na página seguinte, aceite o acordo de licença e selecione o arquivo correspondente (para o Mac OS X x64, no momento em que escrevo esta instrução, o arquivo do último update disponível é o jdk-8u25-macosx-x64.dmg. Veja a imagem seguinte:

2º Passo: Instalação

Aguarde o final do download, que pode demorar um pouco, dependendo da velocidade de sua conexão com a Internet, pois o tamanho do arquivo mencionado é de 209 MB, e execute o arquivo dmg de instalação. O processo é praticamente automático.

3º Passo: Verificação

Para conferir a instalação, abra um Terminal, e digite java -version. Deverão ser exibidas as strings de controle de versão do JRE, conforme a figura abaixo.
